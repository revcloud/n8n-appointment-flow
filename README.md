# Post-Lead Experience

## Main Branch

- **Branch**: `main`

## Setup

1. **Navigate to project directory**
   ```bash
   cd n8n-flow-experience
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm start
   ```

4. **Build for production**
   ```bash
   npm run build
   ```


## Project Structure

```
n8n-flow-experience/
├── src/
│   ├── api/              # API integrations
│   ├── components/       # React components
│   ├── experiences/      # Experience pages
│   └── utils/           # Utility functions
└── public/               # Static assets
```

## URL Parameters

- `plid`: Post Lead ID
- `quiz_address`: Quiz user address
- `quiz_phone`: Quiz user phone
- `quiz_email`: Quiz user email
- `quiz_name`: Quiz user name
- `prepop_address`: Prepop user address
- `prepop_phone`: Prepop user phone
- `prepop_email`: Prepop user email
- `prepop_name`: Prepop user name
- `bedsCount`: No of beds info
- `bathsCount`: No of baths info
